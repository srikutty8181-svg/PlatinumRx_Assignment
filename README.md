# PlatinumRx Data Analyst Assignment

## 1. Project Overview
This project demonstrates core data analysis skills across **SQL**, **Excel/Spreadsheets**, and **Python**.  
It covers database creation, querying, time-based analysis, and basic programming logic.

## 3. SQL Details

### Hotel System
- **Tables:** `users`, `bookings`, `items`, `booking_commercials`  
- **Queries:**  
  1. Last booked room  
  2. Billing in November 2021  
  3. Bills greater than 1000  
  4. Most and least ordered items by month  
  5. Second highest bill  

### Clinic System
- **Tables:** `clinic_sales`, `expenses`  
- **Queries:**  
  1. Revenue by sales channel  
  2. Monthly profit/loss  
  3. Other analysis queries as required  

> All SQL scripts are in the `SQL/` folder.

---

## 4. Python Scripts

1. **01_Time_Converter.py**  
   - Converts integer minutes into "X hours Y minutes" format.  
   - Example: `130 → 2 hours 10 minutes`

2. **02_Remove_Duplicates.py**  
   - Takes a string input and removes duplicate characters.  
   - Example: `"hello" → "helo"`

> All Python scripts are in the `Python/` folder.

---

## 5. Spreadsheet (Excel)

- **Ticket_Analysis.xlsx** contains:
  - `ticket` and `feedbacks` sheets  
  - VLOOKUP formulas to fetch data  
  - Time analysis using `INT` and `HOUR` to find tickets created & closed same day/hour  

> All spreadsheet files are in the `Spreadsheets/` folder.
